## Getting Started

First, install the back-end dependencies:

```bash
cd api
```

```bash
npm install
```

Then, install the front-end dependencies:

```bash
cd app
```

```bash
npm install
```

Then, run both back and front end:

```bash
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
